# Manual del portal

## Objetivo

El portal permite acceder de forma centralizada a herramientas digitales del Servicio de Farmacia, sin modificar su independencia técnica.

## Cómo usar el portal

1. Abrir la página principal del portal.
2. Identificar la sección funcional correspondiente.
3. Seleccionar la herramienta deseada.
4. Usar:
   - **Acceder** para entrar en la herramienta.
   - **Documentación** para consultar información funcional/técnica.
   - **Código** cuando exista repositorio asociado.

## Herramientas incluidas en esta fase

- PIDEFAR
- Guardias Farmacia
- Sesiones Clínicas

## Notas para usuarios

- Si un enlace no está disponible todavía, aparecerá como pendiente.
- La información de estado y tipo puede actualizarse conforme avance la validación de cada herramienta.
